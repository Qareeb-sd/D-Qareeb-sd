# موقع لوحة الإدارة «قريب»

لوحة الإدارة **موقع ويب فقط** (ليست تطبيقاً)، مستقلّة عن تطبيقَي العميل والسائق —
لا تُحمَّل داخل أي APK. منها تُدير المنصّة: **اعتماد/رفض طلبات السائقين**، طلبات
تعبئة المحفظة، التسعير، العمولة، الحساب البنكي، الملخّص المالي، وتنبيهات الطوارئ.

## التشغيل محلياً (الأسرع للاستخدام الآن)
من مجلد `qareeb-web` (مع وجود `.env` بمفاتيح Supabase):
```
npm run dev:admin
```
ثم افتح الرابط الظاهر (مثل `http://localhost:5173`) في المتصفح.

## النشر على Cloudflare Pages (موصى — رابط دائم بدون خادم)
لوحة الأدمن **موقع ثابت (SPA)** يتصل بـ Supabase مباشرة، فينشر على Cloudflare Pages بسهولة.

**من لوحة Cloudflare → Workers & Pages → Create → Pages → Connect to Git:**
1. اربط مستودع GitHub: `Qareeb-sd/D-Qareeb-sd`.
2. **Production branch:** `claude/development-continuation-nxfgub` (أو `main` بعد الدمج).
3. **Build settings:**
   - Framework preset: **None**
   - **Root directory (advanced):** `qareeb-web`
   - **Build command:** `npm run build:admin`
   - **Build output directory:** `dist`
4. **Environment variables** (Production + Preview):
   ```
   VITE_SUPABASE_URL       = https://xxxx.supabase.co
   VITE_SUPABASE_ANON_KEY  = eyJ…
   ```
   (مفتاح الخرائط غير مطلوب للأدمن.) الإصدار Node يُقرأ من `.nvmrc` (20).
5. **Save and Deploy.** بعد البناء يعطيك رابطاً مثل `qareeb-admin.pages.dev`.
6. **نطاق مخصّص (اختياري):** Custom domains → أضِف مثل `admin.qareeb.sd`.

أي دفع جديد للفرع يُعيد النشر تلقائياً. توجيه SPA مضبوط عبر `public/_redirects`.

## أو بناء يدوي محلي
```
npm run build:admin      # يُنتج dist/ للرفع على أي استضافة ثابتة
```

> انشره على **نطاق فرعي منفصل** (مثل `admin.qareeb.sd`) — ليس نفس موقع العملاء.

## الدخول
- الرابط الجذر للموقع = **شاشة دخول الإدارة**.
- سجّل دخول برقم هاتف **دوره `admin`** في قاعدة البيانات + كلمة السر.
- من ليس أدمن يُمنع (الحماية مفروضة أيضاً عبر RLS ودوال Postgres، لا الواجهة فقط).

> إن رفض الدخول لحساب ضبطته عبر SQL: تأكّد أن الحساب **سجّل دخوله مرّة من التطبيق**
> (حتى يُنشأ في نظام مصادقة Supabase) ثم عُيِّن دوره `admin` على نفس الصف.

## الأقسام
| القسم | الوظيفة |
|---|---|
| نظرة عامة | رحلات اليوم · سائقون متصلون · تعبئات معلّقة · الملخّص المالي |
| الطلبات | **اعتماد/رفض السائقين** (مع عرض الوثائق والصور) + طلبات التعبئة |
| السائقون | كل السائقين وحالتهم |
| الرحلات | أحدث الرحلات وحالاتها |
| الإعدادات | التسعير لكل مركبة · العمولة · Surge · الحساب البنكي |
