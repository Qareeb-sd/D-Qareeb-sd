# صور المركبات

ضع هنا صور المركبات الحقيقية (PNG بخلفية شفافة يُفضّل). الأسماء المتوقعة
(انظر `src/data/services.ts`):

| الملف | الخدمة | ملاحظة |
|------|--------|--------|
| `standard.png` | قريب عادي | يارس أبيض |
| `commute.png`  | ترحيل | نفس صورة التاكسي (أبيض) |
| `vip.png`      | قريب VIP | مرسيدس مايباخ |
| `hiace.png`    | هايس | 14 راكب فضي |
| `amjad.png`    | أمجاد | **ناقصة — تنتظر صورة الداماس الأزرق الحقيقية** |
| `open.png`     | مشوار مفتوح | أبيض |
| `rickshaw.png` | ركشة | الركشة السودانية |

إن غابت الصورة يعرض التطبيق بديلاً رسومياً تلقائياً (انظر
`src/components/VehicleImage.tsx`)، فلا يتعطّل شيء.
