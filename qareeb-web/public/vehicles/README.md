# صور المركبات

ضع هنا صور المركبات الحقيقية (PNG بخلفية شفافة يُفضّل). الأسماء المتوقعة
(انظر `src/data/services.ts`):

| الملف | الخدمة | المرجع |
|------|--------|--------|
| `standard.png` | قريب عادي | يارس أبيض |
| `commute.png`  | ترحيل | يارس أبيض (نفس العادي) |
| `vip.png`      | قريب VIP | مرسيدس مايباخ — **ما زالت الصورة الحقيقية ناقصة** |
| `hiace.png`    | هايس | تويوتا هايس فضي (14 راكب) |
| `amjad.png`    | أمجاد | دايو داماس **أزرق** |
| `open.png`     | مشوار مفتوح | سيارة بيضاء |
| `rickshaw.png` | ركشة | توك‑توك سوداني أسود/ذهبي |

## البديل التلقائي

إن غابت الصورة يعرض التطبيق **رسم SVG مميّزاً لكل نوع** (وليس صورة عامّة) —
مُصمَّم ليطابق المركبة الحقيقية بالشكل واللون. انظر:

- `src/components/VehicleArt.tsx` — الرسوم لكل نوع (`sedan/taxi/luxury/van/microbus/rickshaw`)
- `src/components/VehicleImage.tsx` — يجرّب الصورة أولاً ثم يسقط للرسم
- الشكل واللون لكل خدمة معرّفان في `src/data/services.ts` (`art` و`tint`)

## لإضافة الصور الحقيقية

1. سمِّ كل ملف بالاسم أعلاه وضعه في هذا المجلد.
2. الصورة ستُعرض تلقائياً بدل الرسم (لا تعديل كود مطلوب).
3. احذف/تجاهل الرسم البديل — يبقى احتياطياً فقط.
