# صور المركبات

ضع هنا صور المركبات الحقيقية (PNG بخلفية شفافة يُفضّل). الأسماء المتوقعة
(انظر `src/data/services.ts`):

| الملف | الخدمة | المرجع |
|------|--------|--------|
| `ladies.png`   | قريب نسائي | سيارة وردية · سائقة |
| `amjad.png`    | أمجاد | دايو داماس **أزرق** |
| `hiace.png`    | هايس | تويوتا هايس فضي (14 راكب) |
| `rickshaw.png` | ركشة | توك‑توك سوداني أسود/ذهبي |
| `open.png`     | مشوار مفتوح | سيارة (أي نوع) |
| `tow.png`      | سحاب | سطحة/شاحنة إنقاذ بيضاء |

## البديل التلقائي

إن غابت الصورة يعرض التطبيق **رسم SVG مميّزاً لكل نوع** (وليس صورة عامّة).
انظر:

- `src/components/VehicleArt.tsx` — الرسوم (`sedan/ladies/van/microbus/rickshaw/tow`)
- `src/components/VehicleImage.tsx` — يجرّب الصورة أولاً ثم يسقط للرسم
- الشكل واللون لكل خدمة في `src/data/services.ts` (`art` و`tint`)

## لإضافة الصور الحقيقية

سمِّ كل ملف بالاسم أعلاه وضعه هنا — يُعرض تلقائياً بدل الرسم، بلا تعديل كود.
